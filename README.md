## 🎯 Features

### Primary Search Engines
- 🇨🇳 **Baidu** - China's largest search engine (Primary for Chinese queries)
- 🌍 **Google** - World's most accurate engine (Primary for English queries)
- 🔍 **12 Total Engines** - 360, DuckDuckGo, Qwant, Bing, etc.

### Core Capabilities
- ⏰ **Real-time Search** - Today/24h/7d/30d time filtering
- 🛡️ **Ad Filtering** - 95%+ accuracy, auto-detect & remove
- 🧪 **Content De-toxication** - 90%+ accuracy, filter fake news
- 📊 **Multi-engine Aggregation** - Search 3 engines simultaneously
- 📝 **AI Summaries** - Auto-generated key insights
- 🔒 **Source Verification** - Prioritize authoritative sources
- ⚠️ **4-level Warning System** - Trusted/Questionable/Suspicious/Toxic

## 🚀 Installation

### Via ClawHub (Recommended)
```bash
clawhub install smart-web-search
```

### Manual Installation
```bash
git clone https://github.com/davidme6/openclaw-skills.git
cp -r openclaw-skills/skills/smart-web-search ~/.openclaw/skills/
```
