## 🚀 Quick Start

### Install Skills

```bash
# Via ClawHub
clawhub install <skill-name>

# Or manual install
git clone https://github.com/davidme6/openclaw-skills.git
cp -r openclaw-skills/skills/<skill-name> ~/.openclaw/skills/
```

### Use Skills

Use natural language directly in OpenClaw:
```
"Search XXX"
"Search for XXX"
"Help me find XXX"
```

### Skill Structure

```

└── tools/            # Tool scripts (optional)
```
